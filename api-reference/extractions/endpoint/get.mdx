---
title: "Get Extraction Schemas"
openapi: "openapi-extractions GET /v1/extractions"
---

The **Get Extraction Schemas** endpoint retrieves a list of all extraction schemas available in the ParDocs API. Extraction schemas define the structure of the data you can extract from various documents using the API. This endpoint is useful for understanding the data formats and fields that can be extracted, ensuring you can properly process and utilize the data returned by the API.
<RequestExample>
```bash cURL
curl --request GET \
  --url https://api.pardocs.com/v1/extractions \
  --header 'x-api-key: <api-key>'
```
