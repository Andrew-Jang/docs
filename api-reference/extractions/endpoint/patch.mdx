---
title: "Update Extraction"
openapi: "PATCH /v1/extractions/{extraction_id}"
---

You can provide one or more of the following fields to update them, any field not provided will be left unchanged.

<RequestExample>
```bash cURL
curl --request PATCH \
  --url https://api.pardocs.com/v1/extractions/{extraction_id} \
  --header 'Content-Type: application/json' \
  --header 'x-api-key: <api-key>' \
  --data '{
  "extraction_type": "<string>",
  "extraction_data": {}
}'
```
```python Python
import json
import requests

url = "https://api.pardocs.com/v1/extractions/{extraction_id}"
headers = {
    "x-api-key": "<api-key>",
    "Content-Type": "application/json"
}
data = {
    "extraction_type": "<string>",
    "extraction_data": {}
}

response = requests.patch(url, headers=headers, data=json.dumps(data))

print(response.json())
```
