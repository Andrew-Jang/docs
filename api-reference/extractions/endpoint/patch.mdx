---
title: "Update Extraction Schema"
openapi: "openapi-extractions PATCH /v1/extractions/{extraction_id}"
---

The **Update Extraction Schema** endpoint allows you to modify an existing extraction schema identified by `{extraction_id}`. You can update one or more fields of the extraction schema. Any field that is not provided in the request will remain unchanged.

<Note>Note that this process can also be done in the [ParDocs Console](https://www.pardocs.com)</Note>

**Request Body Parameters:**

- `extraction_type` (string, optional): Specifies the document type the schema applies to.
- `extraction_data` (object, optional): Contains the updated schema definition for the extraction.

<RequestExample>
```bash cURL
curl --request PATCH \
  --url https://api.pardocs.com/v1/extractions/{extraction_id} \
  --header 'Content-Type: application/json' \
  --header 'x-api-key: <api-key>' \
  --data '{
  "extraction_type": "<string>",
  "extraction_data": {}
}'
```
```python Python
import json
import requests

url = "https://api.pardocs.com/v1/extractions/{extraction_id}"
headers = {
    "x-api-key": "<api-key>",
    "Content-Type": "application/json"
}
data = {
    "extraction_type": "<string>",
    "extraction_data": {}
}

response = requests.patch(url, headers=headers, data=json.dumps(data))

print(response.json())
```
</RequestExample>
