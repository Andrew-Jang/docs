---
title: "Split Document by Document Type"
openapi: "openapi-documents POST /v1/documents/{document_id}/split"
---

The **Split Document by Document Type** endpoint allows you to split an existing document into multiple documents based on the specified extraction types. This operation is useful for processing documents with different types of content or sections, ensuring that each part is categorized and handled according to its extraction type.

**Request Body Parameters:**

- `extraction_ids` (array of strings): A list of extraction type IDs that specify how the document should be split. Each ID corresponds to a predefined extraction schema.

<RequestExample>
```bash cURL
curl --request POST \
  --url https://api.pardocs.com/v1/documents/{document_id}/split \
  --header 'x-api-key: <api-key>'
  --data '{
  "extraction_ids": [
    "<string>"
  ]
}'
```
```python Python
import json
import requests

url = "https://api.pardocs.com/v1/documents/{document_id}/split"
headers = {
    "x-api-key": "<api-key>",
    "Content-Type": "application/json"
}
data = {"extraction_ids": ["<string>"]}

response = requests.post(url, headers=headers, data=json.dumps(data))

print(response.json())
```
